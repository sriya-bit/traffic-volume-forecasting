# 🚗 Traffic Volume Forecasting

### 📌 Project Overview
This project focuses on predicting urban traffic volume to assist in route optimization and congestion management. It was developed during a mentor-led internship associated with **Uber x MentorMind**, focusing on time-series data and predictive analytics.

### 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn
- **Models Used:** Random Forest Regressor

### 📊 Key Features & Workflow
1. **Data Preprocessing:** Handled missing values, encoded categorical variables, and performed feature scaling.
2. **Feature Engineering:** Extracted time-based features (Hour, Day of Week, Month) and holiday indicators to improve model accuracy.
3. **Exploratory Data Analysis (EDA):** Visualized traffic trends across different time frames to identify peak congestion periods.
4. **Model Training:** Built and tuned a regression model to forecast volume with high precision.
